# Tipos de dados

Conforme o ECMAScript standard, temos 9 tipos de dados

* Data Types
    * Primitive / Primitive value
    * Structural
    * Structural Primitive

## Primitivos

* String
* Number
* Boolean
* undefined
* Symbol
* BigInit

## Estruturais 

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* function

## Primitivo Estructural / Structural Root Primitive

* null
